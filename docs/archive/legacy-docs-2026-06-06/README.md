# Legacy Docs Archive

This folder stores the pre-consolidation documentation set that existed before `docs/` became the single canonical documentation root.

## Contents

- `README.legacy.md`: old root docs entry point.
- `AI_RULES/`: old AI operational rules structure.
- `core/`: old project summary, workflow, structure, blueprint, and DoD documents.

## Status

These files are historical context only.

They must not override:

- active standards in `docs/`;
- accepted ADRs in `docs/adr/`;
- active blueprints in `docs/blueprints/`;
- source code;
- command output or test proof.

Only revive material from this archive through a new blueprint or ADR-backed edit.
