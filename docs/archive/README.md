# Archive

Historical docs live here.

Archive material cannot override active blueprints, ADRs, source code, or command output.

Suggested archive locations:

```text
docs/archive/handoffs-closed/
docs/archive/blueprints-superseded/
docs/archive/evidence-history/
```

Closed handoffs index:

```text
docs/archive/handoffs-closed/README.md
```

Current handoffs remain in:

```text
docs/handoffs/
```

Archived material is read-only context unless the owner explicitly revives it.
